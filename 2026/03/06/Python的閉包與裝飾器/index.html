它和 JavaScript 的 closure 很像，先看一段程式碼來了解作用域。
它的特色是：外部函式雖然執行完了，但內部函式依然會記住當時環境中的變數。

裝飾器（decorator）

裝飾器可以在不修改原本函式內容的前提下，替函式加上額外功能。裝飾器通常會回傳另一個函式。
常見用途是把和核心邏輯無關、但重複出現的程式碼抽離出來，例如記錄時間、權限檢查、日誌紀錄等。
